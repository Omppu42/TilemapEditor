## Running
### CMD
To check if you have Python installed, open terminal and run:
```
python -V
```
Install python if necessary.
<br>
<br>

Download the TilemapEditor. Navigate to the folder in your terminal using:
```
cd folderName
cd ..
```
When you reach '../TilemapEditor', Run:
```
python3 main.py
```
